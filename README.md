# LearnPlaywrightBatch2x
Learning repository for Playwright automation (Batch 2x).

## Contents
- `Basics.js`: JavaScript for loop example (1 to 1000)
- `chapter_01_JavaScript-Basics/`: JavaScript literals, const, let, var examples
- `chapter_02_javascript/`: JavaScript variables, functions, hoisting, templates
- `chapter_03_identifies/`: Conditional statements (if)
- `chapter_04_statements/`: JavaScript statements
- `chapter_05_operators/`: Arithmetic and assignment operators

## Setup
1. Clone the repo:
   ```bash
   git clone https://github.com/VikasBijanalli/LearnPlaywrightBatch2x.git
   ```
2. Install dependencies (for Playwright):
   ```bash
   npm install playwright
   ```
3. Run sample code:
   ```bash
   node Basics.js
   ```
