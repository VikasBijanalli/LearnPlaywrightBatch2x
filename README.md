# LearnPlaywrightBatch2x
Learning repository for Playwright automation (Batch 2x).

## Contents
- `Basics.js`: JavaScript for loop example (1 to 1000)
- `chapter_01_basics/`: Basic JavaScript and Playwright exercises
- `tests/`: Automation test cases
- `reports/`: Test execution reports

## Setup
1. Clone the repo:
   ```bash
   git clone https://github.com/VikasBijanalli/LearnPlaywrightBatch2x.git
   ```
2. Install dependencies (for Playwright):
   ```bash
   npm install playwright
   ```
3. Run sample code:
   ```bash
   node Basics.js
   ```
